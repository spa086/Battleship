"/whatsUp" 
Request model:
{
	userId - int.
}
JSON Request body examples:
{"userId":144}
Returns object: 
{
    state - string enum { "WaitingForStart", "CreatingShips", "YourTurn", "OpponentsTurn" }
	userId - int optional
}


"/createFleet"
Request model:
{
	userId - integer.
	decks - integer array.
}
Example request body JSON:
{"userId":144,"decks":[1,4,5]}
Returns bool - true if player is the 1st, false if he's the 2nd.

"/attack"
Request model:
{
	SessionId - integer.
	Location - integer.
}
Returns object:
{
	Result - string,
	Player1Fleet - array of objects:
	{
		Decks - array of objects: 
		{
			Location - integer.
			Destroyed - boolean.
		}
	}
	Player2Fleet - array of objects:
	{
		Decks - array of objects: 
		{
			Location - integer.
			Destroyed - boolean.
		}
	}
}
"Result" property can be one of the following strings:
    "Hit",
    "Killed",
    "Missed",
    "Win".

"/abort"
Request model:
{
	SessionId - integer.
}
Returns nothing.
