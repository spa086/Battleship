"/whatsUp" 
Request model:
{
	SessionId - integer.
	IsFirstPlayer - boolean (optional).
}
JSON Request body examples:
{"SessionId":144}
{"SessionId":144, "IsFirstPlayer":true}
Returns one of the following strings:
    "WaitingForStart",
    "CreatingShips",
    "YourTurn",
    "OpponentsTurn".


"/createFleet"
Request model:
{
	SessionId - integer.
	Decks - integer array.
}
Example request body JSON:
{"SessionId":144,"Decks":[1,4,5]}
Returns bool - true if player is the 1st, false if he's the 2nd.

"/attack"
Request model:
{
	SessionId - integer.
	Location - integer.
}
Returns object:
{
	Result - string,
	Player1Fleet - array of objects:
	{
		Decks - array of objects: 
		{
			Location - integer.
			Destroyed - boolean.
		}
	}
	Player2Fleet - array of objects:
	{
		Decks - array of objects: 
		{
			Location - integer.
			Destroyed - boolean.
		}
	}
}
"Result" property can be one of the following strings:
    "Hit",
    "Killed",
    "Missed",
    "Win".

"/abort"
Request model:
{
	SessionId - integer.
}
Returns nothing.
