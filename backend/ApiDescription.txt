"/whatsUp" 
Request model:
{
	SessionId - integer.
}
Example request body JSON:
{"SessionId":144}
Returns one of the following strings:
    "WaitingForStart",
    "CreatingShips",
    "YourTurn",
    "OpponentsTurn".


"/createFleet"
Request model:
{
	Decks - integer array.
}
Example request body JSON:
{"Decks":[1,4,5]}
Returns nothing.

"/attack"
Request model:
{
	X - integer,
	Y - integer.
}
Returns nothing.
