"/whatsUp" 
Request model:
{
	SessionId - integer.
	IsFirstPlayer - boolean (optional).
}
JSON Request body examples:
{"SessionId":144}
{"SessionId":144, "IsFirstPlayer":true}
Returns one of the following strings:
    "WaitingForStart",
    "CreatingShips",
    "YourTurn",
    "OpponentsTurn".


"/createFleet"
Request model:
{
	SessionId - integer.
	Decks - integer array.
}
Example request body JSON:
{"SessionId":144,"Decks":[1,4,5]}
Returns bool - true if player is the 1st, false if he's the 2nd.

"/attack"
Request model:
{
	SessionId - integer.
	Location - integer.
}
Returns nothing.

"/abort"
Request model:
{
	SessionId - integer.
}
Returns nothing.
